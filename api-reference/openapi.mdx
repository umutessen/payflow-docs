The PayFlow API follows REST conventions and uses JSON for
all requests and responses. All endpoints require authentication
via the `X-API-Key` header.

**Base URLs:**
- Sandbox: `https://sandbox.payflow.io/v1`
- Production: `https://api.payflow.io/v1`

<Info>
  Use the **sandbox** environment for all testing.
  Sandbox payments don't move real money and
  support all the same endpoints as production.
</Info>

<Note>
  Enter your API key in the field above to use
  the live "Try it" panel on each endpoint.
</Note>
